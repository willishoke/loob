## loob

### A library for digital systems design

#### Components include:

- Gates
- Multiplexers
- Demultiplexers 
- Adders
- Multipliers
- Bitwise operators
- Shift registers
- Flip flops
- ALU
- RAM

all with support for arbitrarily large word sizes.
